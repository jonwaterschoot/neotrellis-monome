# Multi-Screen Test

This script was built for testing the multiple screen emulation in the webapp. It also serves as a compact case study for NeoTrellis color handling and multiscreen setups.

It is still small enough to run without minifying, but a minified version is included here as well for release/testing purposes.

Files in this folder:

- `multiplescreen-test.lua` — main script
- `multiplescreen-test_min.lua` — minified version
- `manifest.json` — script metadata for emulator/catalog discovery
