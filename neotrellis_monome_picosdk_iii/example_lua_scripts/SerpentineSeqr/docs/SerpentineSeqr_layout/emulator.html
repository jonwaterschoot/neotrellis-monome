Serpentine Sequencer: Component Reference

Settings View Controls

  • Row 1 (cols 1–8): Fruit spawn count slider — each step = 2 fruits.
  • Row 1 (cols 11–16): Fruit type toggles — R, B, Y, C, O, P.
  • Row 2 (cols 1–8): Arpeggio lifespan slider.
  • Row 2 (cols 9–16): Arpeggio pool max capacity.
  • Row 3, col 1: Autopilot mode (NON→SEM→AUT).
  • Row 3, col 3: Arp playback order (ORD→RND→UP→DWN).
  • Row 3, cols 10–13: BPM -10, -1, +1, +10.
  • Row 3, col 15: Master brightness step.
  • Row 3, col 16: Monochrome tint cycle.
  • Row 4 (cols 1–7): Scale selection (MAJ MIN PMA PMI DOR LYD CUS).
  • Rows 6–7 (cols 1–7): Root note keyboard.
  • Row 8, col 1 / Tab key: ALT toggle — pause game & view settings.

Live Game Controls

  • Arrow keys: Steer the snake.
  • Click game grid: Point-to-go (SEM auto), or steer by click direction.
  • D-PAD pads (15,7 / 14,8 / 15,8 / 16,8): Hardware directional controls.
  • Fruit colours: Red = shrink+arp, Blue = grow+arp, Yellow = slow+arp, Cyan = chord, Orange = rare arp, Purple = echo.
