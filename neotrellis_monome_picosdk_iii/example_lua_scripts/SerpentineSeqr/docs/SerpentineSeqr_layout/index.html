Serpentine Sequencer: Component Documentation

The settings menu allows extensive, real-time control over the sequencer and game mechanics. When holding or toggling the ALT button, the 16x8 NeoTrellis grid maps to the following interactive parameters.

Row 1: Spawner Engine

  • Spawn Quantity Slider: Controls the maximum number of fruits allowed on the grid simultaneously. Each step represents 2 fruits, capping at 16.
  • Fruit Type Toggles: A palette allowing independent enabling/disabling of fruit drop permutations. Active fruits glow with native RGB.
    • Red: Basic Note & Tail Shrink
    • Blue: Bonus Length & Arp Trigger
    • Yellow: Temp Slow Down & Arp Trigger
    • Cyan: Diatonic Triads / Chords
    • Orange: 33% Chance Arp Trigger
    • Purple: Gravity Echo Bounces

Row 2: Arpeggiator Configuration

  • Arpeggio Lifespan: Sets how many steps the active Arpeggiator sequence lasts before decaying.
  • Arpeggio Pool Capacity: Dictates the maximum buffer size of notes the snake can hold concurrently in historical memory.

Row 3: Timing & Macros

  • Autopilot Mode: Cycles between 3 snake AI navigation states (NON, SEM, AUT).
  • Arpeggio Playback Mode: Cycles the sequence order in which the internal Arp Pool repeats (ORD, RND, UP, DWN).
  • Tempo (BPM) Controls: Increment and decrement buttons for the master clock speed (-10, -1, +1, +10).
  • Master Grid Brightness: Steps through internal LED brightness scaling limits.
  • Monochrome/Tint Overlay: Toggles between rich color display and 8 cinematic monochromatic hue layers.

Row 4: Scale System

  • Scale Selection Tabs: Determines the diatonic scale structure filtering the matrix logic (MAJ, MIN, PMA, PMI, DOR, LYD, CUS).
  • Dynamic LED Numeric Readout Space: The right side is a physical matrix dedicated to rendering text blocks utilizing a 3x5 font pixel-array.

Rows 6 & 7: The Interactive Keyboard

  • Live Controller: Standard Piano chromatic layout. Used for locking the global Root Note of the sequencer, or plotting toggle positions manually when CUSTOM scale mode is enabled.

Global Layout Commands

  • ALT Access Key: The entry/exit toggle. Tapping it quickly activates sticky-mode, holding and releasing enables momentary access.
  • D-PAD Area: 4-way hardware directional keys (UP, DOWN, LEFT, RIGHT) to pilot the snake manually across the simulator grid.
